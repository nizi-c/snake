# 🎨 主题切换功能说明

## 功能概述

贪吃蛇游戏现在支持两套精美主题，可以根据个人喜好和使用场景自由切换。

## 主题风格

### ☀️ 浅色主题 - 明媚温暖柔和
采用温暖明亮的色彩搭配，营造轻松愉悦的游戏氛围。

**配色方案：**
- 🎨 **主色调**：鹅黄色 (#FFF9E6) - 温暖舒适的背景
- 🌟 **强调色**：温暖橙 (#FFB74D) - 活力四射
- 💙 **按钮色**：天蓝色 (#64B5F6) - 清新明快
- 🌿 **成功色**：清新绿 (#81C784) - 生机勃勃
- 📝 **文字色**：深棕色 (#5D4037) - 清晰易读

**适用场景：**
- 白天游玩
- 明亮环境
- 追求活力感

### 🌙 深色主题 - 浅蓝色系
采用清新的浅蓝色系，低饱和度配色更护眼，适合长时间游玩。

**配色方案：**
- 🎨 **主色调**：深蓝灰 (#2C3E50) - 沉稳大气
- 🌟 **强调色**：天蓝色 (#5DADE2) - 清新明快
- 💙 **按钮色**：亮蓝色 (#3498DB) - 活力十足
- 🌿 **成功色**：青绿色 (#52C7B8) - 自然和谐
- 📝 **文字色**：浅灰白 (#ECF0F1) - 柔和护眼

**适用场景：**
- 夜间游玩
- 暗光环境
- 追求护眼效果
- 喜欢低调优雅风格

## 使用方法

### 1. 进入主题设置
在游戏首页，点击右上角的 **⚙️ 主题** 按钮

### 2. 选择主题
- 点击 **浅色主题** 卡片切换到明亮模式
- 点击 **深色主题** 卡片切换到深色模式
- 当前使用的主题会显示 **✓ 当前使用** 标识

### 3. 自动应用
主题切换后会立即生效，并应用到所有页面：
- ✅ 首页
- ✅ 游戏页面
- ✅ 角色选择页面
- ✅ 历史记录页面
- ✅ 主题设置页面

### 4. 持久化保存
选择的主题会自动保存，下次打开应用时会自动加载上次使用的主题。

## 技术实现

### 主题管理器
使用单例模式的 `ThemeManager` 统一管理主题：
- 主题加载和保存
- 主题切换通知
- 跨页面主题同步

### 响应式更新
所有页面通过 `@State` 响应主题变化，实现实时更新。

### 本地存储
使用 HarmonyOS 的 `preferences` API 持久化保存主题设置。
